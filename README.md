# Link App

Font-End: React, Redux
Back-End: NodeJS, Express, MongoDB

## Setup

```
$ npm install
```

## Running in dev mode

```
$ npm run dev
```

## Build client (production)

```
$ npm run build
```

## Running client

```
$ npm start
```

## Remove dist folder

```
$ npm run clean
```
